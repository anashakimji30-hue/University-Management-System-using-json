# 🎓 University Management System (JSON-Based)

A comprehensive data management project that simulates a real-world university system using structured JSON.

---

## 🚀 Features

- 📊 Student Performance Tracking (GPA)
- 📚 Course & Attendance Management
- 🏫 Department Analytics
- 💰 Financial Summary (Fees, Budget)
- 🔍 Search Functionality

---

## 💡 Key Learning

- Handling complex nested JSON structures  
- Modeling real-world relationships without a database  
- Building logical connections between entities  
- Creating a clean and interactive dashboard  

---

## 🛠️ Tech Used

- HTML  
- CSS  
- JavaScript  
- JSON
